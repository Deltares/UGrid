inlinestatic

This function adjusts the values in an input array based on the difference between the provided start_index and the start_index stored in the specified NetCDF variable's attribute named "start_index". If the attribute is not found, no adjustments are made to the array.

Template Parameters
TThe data type of the input array values.
Parameters
[in]varThe NetCDF variable from which to retrieve the "start_index" attribute.
[in]start_indexThe desired start index for the array.
[in]values_sizeThe size of the input array values.
[in,out]valuesThe input array to which the offset is applied.
